## 3. Sectie: Waarvoor kan je support aanvragen?

```text
[Website werkt niet] [Bug/foutmelding] [Formulier werkt niet]
[Content aanpassen] [Tracking/analytics] [Hosting/domein]
```

Korte uitleg:

> Gebruik dit formulier voor vragen of problemen rond je website, formulieren, tracking, hosting, domein of content.

## 4. Sectie: Prioriteiten

```text
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ Normaal      │ │ Dringend     │ │ Noodgeval    │
│ Gewone vraag │ │ Leadverlies  │ │ Site offline │
└──────────────┘ └──────────────┘ └──────────────┘
```

Noodgeval disclaimer:

> Een ticket met prioriteit “Noodgeval” wordt sneller bekeken, maar betekent geen automatische 24/7 support tenzij dit contractueel is afgesproken.

## 5. Sectie: Wat gebeurt er na je aanvraag?

```text
01 Je dient je ticket in
02 Je krijgt een bevestiging met ticketnummer
03 Wij bekijken je aanvraag tijdens de kantooruren
04 We nemen contact op of lossen het probleem op
```
